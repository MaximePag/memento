Commandes Terminal Linux

Commandes GIT Description
$cd [Option] [Dossier]

Navigation dans l’arborescence des répertoires.

Uiliser "$cd -" permet de revenir au répertoire précédent.
$ls [Option] [Dossier] Liste du contenu du répertoire.
$mkdir [Option] [Dossier] Créer un dossier dans le répertoire actuel.
$rmdir [Option] [Dossier]

Supprime un dossier vide dans le répertoire actuel.

Utiliser "$rmdir -r [Dossier]" supprime le dossier et son contenu.
$cp [Option] [Source] [Cible]

Copie un fichier ou un dossier de la source jusqu'à la cible.

Si la commande est sans option cela copiera par défaut un dossier sans son contenu.

Utiliser "$cp -r [Source] [Cible]" copie le dossier et son contenu.
