Commandes Terminal Linux

Commandes Linux Description
$cd [Option] [Dossier]

Navigation dans l’arborescence des répertoires.

Uiliser "$cd -" permet de revenir au répertoire précédent.
$ls [Option] [Dossier] Liste du contenu du répertoire.
$mkdir [Option] [Dossier] Créer un dossier dans le répertoire actuel.
$rmdir [Option] [Dossier]

Supprime un dossier vide dans le répertoire actuel.

Utiliser "$rmdir -r [Dossier]" supprime le dossier et son contenu.
$cp [Option] [Source] [Cible]

Copie un fichier ou un dossier de la source jusqu'à la cible.

Si la commande est sans option cela copiera par défaut un dossier sans son contenu.

Utiliser "$cp -r [Source] [Cible]" copie le dossier et son contenu.

Commandes Terminal GIT

Catégorie Commandes GIT Description
Configuration

$git config --global user.name [Nom du compte GitHub]

$git config --global user.email [EMail du compte GitHub]

 Ces deux commandes permettent de configurer GIT dans le terminal.
Statut des fichiers $git status Statut des fichiers.
Branch $git branch -a Lister les branchs sur la branch courante.
$git branch [Nom de la branch] Créer une nouvelle branch.
$git checkout [Nom de la branch] Basculer sur la branch.
$git branch -d [Nom de la branch]

Supprime la branch si elle est locale seulement.

Si la branch est présente sur un repo distant utilisez:

$git origin --delete [Nom de la branch distante]
Commit

$git add [fichier] [fichier]...

$git commit -m "[Message du commit]"

Sélectionne les fichiers pour le commit.

Créer un commit des fichiers précédemment ajoutés.
$git push -u origin [Nom de la branch] Envoi les commits vers le repo distant sur la branch mentionée.
$git pull -u origin [Nom de la branch] Met à jour les fichiers locaux depuis le repo distant et la branch mentionée.
Remote $git remote add origin [url du repo]

Créer un lien entre le dépôt local et le repo distant.

Exemple:

$git remote add origin https://github.com/[Nom du compte]/[Nom du repo].git
$git remote set-url origin [url du repo] Change l'url du dépôt local vers le repo distant.
Initialiser un projet
  1. Créer un repo sur GitHub
  2. Sur le terminal: Aller à l'emplacement du dépôt local
  3. $git init
  4. $git add [fichier]...
  5. $git commit -m "[Message du commit]"
  6. $git remote add origin [url du repo]
  7. $git push -u origin [Nom de la branch]
  8. Saisisser Compte et Mot de passe
